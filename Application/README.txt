Ferramentas:
	Servidor: NodeJS - http://nodejs.org/
	
	
	Database: SQLite Database Browser - http://sqlitebrowser.sourceforge.net/
		Esta ferramenta serve para auxiliar na visualização da Base de Dados
		Embora não seja fulcral é aconselhado. Outra ferramente de igual
		utilidade pode ser usada.
	
Passos:
	1 - Instalar o NodeJS
	2 - Abrir linha de comandos no directorio dado (mesmo deste ficheiro)
	3 - npm install
	4 - npm link
	5 - supervisor server
	6 - No browser ir para "localhost:81"
	7 - Se desejar abrir o ficheiro database.db para visualização da base de dados
	
	
TroubleShoot	
	Apenas em caso de problemas instalar todos os módulos individualmente
		-npm install express
		-npm install sqlite3
		-npm install moment
		-npm install supervisor
		-npm install ejs
		-supervisor server
		
	Verificar que a firewall não se encontra a bloquear o NodeJS
